> DrunkConcatenate Command Line utility function to display the contents of files.

## Features

---

- View a single file

  - `drunkcat file_name`
  - `drunkcat example.txt`

- View multiple files

  - `drunkcat example1.txt example2.txt`

- Redirect output to a file
  - `drunkcat fileA.txt fileB.txt > resultAB.txt`

### Flags

---

- -n: add line number
